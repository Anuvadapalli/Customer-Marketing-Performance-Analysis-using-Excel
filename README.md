# Customer-Marketing-Performance-Analysis-using-Excel
Imported raw data into Excel, performed ETL using Power Query (data cleaning and transformation), and built Pivot Tables to analyze customer and marketing performance. 
Data Processing & Analysis Workflow
Data Extraction: Imported raw customer and marketing data into Excel
Data Transformation (ETL): Cleaned and transformed data using Excel/Power Query (handled missing values, removed duplicates, formatted data)
Data Analysis: Created Pivot Tables and charts to analyze trends and performance
Visualization: Built an interactive dashboard to present key insights
